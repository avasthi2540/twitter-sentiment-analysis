# Twitter Sentiment Analysis

## 📌 Project Overview
This project is a Machine Learning-based application that analyzes text data and classifies it into Positive or Negative sentiment. It helps understand user opinions using Python and basic Natural Language Processing techniques.

---

## 🚀 Features
- Classifies text as Positive or Negative sentiment  
- Uses Machine Learning (Naive Bayes Algorithm)  
- Performs text preprocessing like cleaning and tokenization  
- Takes real-time user input for prediction  
- Displays model accuracy  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- Machine Learning (Naive Bayes)  
- Basic NLP (Text Processing)  

---

## 📂 How to Run

Install dependencies:
pip install pandas scikit-learn

Run the project:
python sentiment_analysis.py

---

## 💡 Example Output

Enter text: I love this product  
Predicted Sentiment: Positive  

Enter text: This is worst experience  
Predicted Sentiment: Negative  

---

## 🎯 Objective
To build a simple sentiment analysis model that can classify text data and understand user sentiment using machine learning techniques.
